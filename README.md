# 🏠 House Price Prediction using Machine Learning

## Overview

This project uses Machine Learning to predict house prices based on various property features such as area, number of bedrooms, bathrooms, and other housing attributes.

The model is trained on historical housing data and helps estimate property prices accurately.

---

## Features

* Predict house prices based on user inputs
* Data preprocessing and feature engineering
* Machine Learning regression model
* Model performance evaluation
* User-friendly prediction interface

1. Load the housing dataset.
